This guide shows how to take an existing chart, open it in the ChartOut Store, and preview it on products. ChartOut works with your existing chart library, it doesn't replace it.

If you have not installed yet, see [Python installation](/docs/python) or [JavaScript installation](/docs/javascript).

## 1. Start with a chart

Use any supported chart library. The preview below uses an example Vega-Altair chart.

## 2. Connect your chart

How you connect depends on where your chart runs.

<Tabs items={['Python', 'JavaScript']}>
  <Tab value="Python">
    Pass your chart to `chartout.Store`. In notebook environments, evaluating the Store object as the last expression displays the widget inline.

    ```python
    import chartout

    store = chartout.Store(chart)
    store  # renders the widget inline in a notebook environment
    ```

    See [Environments](/docs/python/environments) for host-specific notes and [Example usage](/docs/python/example-usage) for `Store`, `item`, and `Cart` patterns.
  </Tab>
  <Tab value="JavaScript">
    Render your chart to an `<svg>`, create a ChartOut model, mount the widget, then call `openWithViz` to open the store.

    ```tsx
    import { createChartoutModel } from 'chartout'
    import { ChartoutWidget } from 'chartout/react'
    import { openWithViz } from 'chartout/store'

    const model = createChartoutModel({})

    <ChartoutWidget model={model} style={{ width: '100%' }} />

    await openWithViz(model, svg, 'My chart title')
    ```

    See [Integrations](/docs/javascript/integrations) for React, Vue, Svelte, and vanilla JS mount patterns, and [Example usage](/docs/javascript/example-usage) for `openWithViz`, `openWithItem`, and `openWithCart`.
  </Tab>
</Tabs>
