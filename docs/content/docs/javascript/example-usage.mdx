There are three ways to open the store, depending on how much you want to decide for the user. Any chart library works as long as it can produce an `<svg>` element, the examples below show Observable Plot and ECharts (with its SVG renderer). Each example first renders a chart, then opens the store with it.

## openWithViz

Use `openWithViz` when product selection belongs to the user. The widget opens on the default product and lets them switch to any other.

Good for: **"print this" buttons**, live chart previews, exploratory dashboards.

<Tabs items={['Observable Plot', 'ECharts']}>
  <Tab value="Observable Plot">
    ```ts
    import * as Plot from '@observablehq/plot'

    const svg = Plot.plot({
      marks: [
        Plot.rectY(data, Plot.binX({ y: 'count' }, { x: 'eruptions', fill: 'steelblue' })),
    ```

    <ChartOutput sessionId="js-viz-plot" library="plot" pattern="vizlike" />

    ```tsx
    import { createChartoutModel } from 'chartout'
    import { ChartoutWidget } from 'chartout/react'
    import { openWithViz } from 'chartout/store'

    const model = createChartoutModel({})
    await openWithViz(model, svg, 'Old Faithful Eruptions')

    <ChartoutWidget model={model} style={{ width: '100%' }} />
    ```

## openWithItem

Use `openWithItem` when your UI decides the product, not the user. The store opens with that product pre-selected and the chart already applied.

Good for: **product-specific "print on this mug" actions**, embedded product pages, upsell flows.

<Tabs items={['Observable Plot', 'ECharts']}>
  <Tab value="Observable Plot">
    ```ts

      inset: 10,
      marks: [
        Plot.density(data, { x: 'waiting', y: 'eruptions', stroke: 'steelblue' }),
        Plot.dot(data, { x: 'waiting', y: 'eruptions', fill: 'currentColor', r: 1.5 }),
      ],
    })
    ```

### Product IDs

```ts
import { products } from 'chartout/store'

const catalogue = await products()
// catalogue[0].id === 'canvas_10x10'
// catalogue[0].placements[0].id === 'default'
// catalogue[0].placements[0].print_ratio === '1:1'
```

Use the `print_ratio` to render your chart at proportions that suit the product's print area.

## openWithCart

Use `openWithCart` when you know exactly what the user wants to order. The store opens on the cart review screen, no product browsing needed. Each product can carry a different chart.

Good for: **bundles**, dashboards with multiple charts, "order all my charts" flows.

<Tabs items={['Observable Plot', 'ECharts']}>
  <Tab value="Observable Plot">
    ```ts
    import * as Plot from '@observablehq/plot'

    // a different chart per product
    const canvasSvg = Plot.plot({
      marks: [
        Plot.density(data, { x: 'waiting', y: 'eruptions', stroke: 'steelblue' }),
        Plot.dot(data, { x: 'waiting', y: 'eruptions', fill: 'currentColor', r: 1.5 }),
      ],
    })
    const mugSvg = Plot.plot({
      marks: [
        Plot.rectY(data, Plot.binX({ y: 'count' }, { x: 'eruptions', fill: 'steelblue' })),
        Plot.ruleY([0]),
      ],
    })
    const mousepadSvg = Plot.plot({
      axis: null,
      color: { scheme: 'RdBu' },
      marks: [Plot.raster(values, { width: cols, height: rows, x1: -2, x2: 2, y1: -2, y2: 2 })],
    })
    ```

    <ChartOutput sessionId="js-cart-plot" library="plot" pattern="cart" />

    ```tsx
    import type { CartItem } from 'chartout'
    import { createChartoutModel } from 'chartout'
    import { ChartoutWidget } from 'chartout/react'
    import { openWithCart, svgToBytes } from 'chartout/store'

    const model = createChartoutModel({})
    const [canvasBytes, mugBytes, mousepadBytes] = await Promise.all([
      svgToBytes(canvasSvg),
      svgToBytes(mugSvg),
      svgToBytes(mousepadSvg),
    ])

    openWithCart(model, [
      { id: 'canvas_10x10',       name: 'My Canvas (10″×10″)', quantity: 1, placements: [{ id: 'default', content: canvasBytes }] },
      { id: 'mug_black_11oz',     name: 'My Mug (11 oz)',      quantity: 1, placements: [{ id: 'default', content: mugBytes }] },
      { id: 'mousepad_white_8x7', name: 'My Mousepad (8″×7″)', quantity: 2, placements: [{ id: 'default', content: mousepadBytes }] },
    ] satisfies CartItem[])

    <ChartoutWidget model={model} style={{ width: '100%' }} />
    ```

    <StoreOutput sessionId="js-cart-plot" pattern="cart" />
  </Tab>
  <Tab value="ECharts">
    ```ts
    import * as echarts from 'echarts'

    // render any ECharts option to an <svg> via the SVG renderer
    function echartsSvg(option: echarts.EChartsOption, width: number, height: number) {
      const el = document.createElement('div')
      // animation: false draws the final frame synchronously so the export isn't empty
      echarts.init(el, null, { renderer: 'svg', width, height }).setOption({ animation: false, ...option })
      return el.querySelector('svg')!
    }

    const canvasSvg   = echartsSvg(scatterOption, 800, 800)   // scatter
    const mugSvg      = echartsSvg(histogramOption, 800, 320) // bar histogram
    const mousepadSvg = echartsSvg(heatmapOption, 800, 660)   // heatmap
    ```

    <ChartOutput sessionId="js-cart-echarts" library="echarts" pattern="cart" />
