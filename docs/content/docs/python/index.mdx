Install chartout and place your first order from a notebook.

ChartOut works inside any Jupyter-compatible environment, including JupyterLab, Jupyter Notebook, VS Code, Marimo, and Positron. It requires Python 3.11 or higher.

## Installation

Install chartout from PyPI using your preferred package manager. Both uv and pip are supported.

<Tabs items={['uv', 'pip']}>
  <Tab value="uv">
    ```bash
    uv add chartout
    ```
  </Tab>
  <Tab value="pip">
    ```bash
    python -m pip install chartout
    ```
  </Tab>
</Tabs>

## Verifying the installation

After installing, confirm everything is working by checking the version:

```python
import chartout
print(chartout.__version__)
```

## Dependencies

ChartOut has exactly two hard dependencies. [`anywidget`](https://anywidget.dev) provides the bridge between Python and the browser widget, and [`traitlets`](https://traitlets.readthedocs.io) handles the typed state that keeps them in sync. That is all. No chart library is required. ChartOut only uses the libraries you pass into it, following the "support all, depend on none" design.

Currently supported chart libraries are Altair and matplotlib. Pass a chart or figure directly to `chartout.Store` and ChartOut handles the conversion to a print-ready image. Support for vgplot, Plotly, and other libraries is planned. Contributions are welcome.
