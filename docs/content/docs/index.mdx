**ChartOut** converts data visualizations into print-ready products like mugs, canvases, posters and mousepads. You pass a chart, the widget lets the user preview it on a 3D product model, adjust the placement, and check out.

## Highlights

- Works with Altair, matplotlib, Plotly, pyobsplot, Observable Plot, D3, ECharts, and any library that produces SVG
- Three integration patterns: let the user pick a product, pre-select one for them, or build the entire cart upfront
- Real-time 3D preview with placement controls
- Ships to anywhere Printful delivers

## Quick example

import { ChartPreview, StorePreview } from '@/components/demo'

```python
import altair as alt
from altair.datasets import data

source = data.movies.url

chart = alt.Chart(source).mark_bar().encode(
    alt.X("IMDB Rating:Q").bin(maxbins=20).scale(domain=[1, 10]),
    alt.Y('count()'),
    alt.Color("IMDB Rating:Q").bin(maxbins=20).scale(scheme='pinkyellowgreen')
)
chart
```

<ChartPreview />

```python
import chartout

store = chartout.Store(chart)
store  # renders the widget inline in a notebook environment
```

<StorePreview />

From there you can browse products, preview your chart, adjust placement, and check out.
