## What is chartout?

**chartout** lets you convert data visualizations — Altair, matplotlib, Plotly, and pyobsplot charts — into print-ready textures for physical products like mugs, posters, canvases, and t-shirts.

It works in two parts:

- A **Python package** that runs inside Jupyter and lets you browse products, preview your chart on them, and add them to a cart.
- A **JavaScript widget** that renders the interactive store UI directly in your notebook.

## How it works

```python
import chartout

store = chartout.Store()
store.add(my_altair_chart)
store
```

That's it. The store opens inline in the notebook, you pick a product, and chartout handles the rest — from image generation to checkout.
