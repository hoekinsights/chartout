## `chartout.Store`

The main entry point. Renders as an interactive widget in Jupyter.

```python
store = chartout.Store(api_url="https://api.chartout.io")
```

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `api_url` | `str` | `"https://api.chartout.io"` | Base URL of the chartout API |

### `store.add(chart, placement=None)`

Add a chart to the store as an active texture.

```python
store.add(my_chart)
store.add(my_chart, placement=chartout.PlacementPosition(x=0.5, y=0.5, scale=1.2))
```

### Supported chart types

- `altair.Chart`
- `matplotlib.figure.Figure`
- `plotly.graph_objects.Figure`
- `pyobsplot` charts

## `chartout.PlacementPosition`

Fine-grained control over how a chart is placed on a product texture.

```python
pos = chartout.PlacementPosition(x=0.5, y=0.5, scale=1.0, rotation=0)
```

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `x` | `float` | `0.5` | Horizontal center (0–1) |
| `y` | `float` | `0.5` | Vertical center (0–1) |
| `scale` | `float` | `1.0` | Scale multiplier |
| `rotation` | `float` | `0` | Rotation in degrees |
