## Overview

The chartout widget is built with [anywidget](https://anywidget.dev/), which lets it render a React app directly inside Jupyter cells. The Python `Store` class syncs state with the widget over a message channel.

## Synced properties

The widget exposes four synced properties between Python and the browser:

| Property | Type | Description |
|----------|------|-------------|
| `cart` | `list` | Current cart items |
| `active_texture` | `str \| null` | Base64-encoded texture image |
| `active_item` | `str \| null` | Currently selected product item ID |
| `init_viz` | `bool` | Whether to initialise the visualisation |

## Development

The widget bundle is served from the `bundle/` directory and built with esbuild. To run in development mode:

```bash
pixi run npm-dev   # chartout-app
pixi run lab       # Jupyter
```

Both hot-reload as you edit.

## Widget entry points

- `Widget.tsx` — the Jupyter widget (anywidget integration)
- `website.tsx` — the standalone demo site
