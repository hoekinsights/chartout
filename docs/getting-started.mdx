## Prerequisites

- Python 3.10+
- A running Jupyter environment (JupyterLab or Jupyter Notebook)
- [Pixi](https://prefix.dev/docs/pixi/overview) for dependency management (recommended)

## Installation

```bash
pixi add chartout
```

Or with pip:

```bash
pip install chartout
```

## Your first chart product

Open a Jupyter notebook and run:

```python
import altair as alt
import pandas as pd
import chartout

# Create any chart
chart = alt.Chart(pd.DataFrame({'x': range(10), 'y': range(10)})).mark_line().encode(
    x='x', y='y'
)

# Open the store
store = chartout.Store()
store.add(chart)
store
```

The chartout widget opens inline. From there you can:

1. Browse available products (mug, poster, canvas, t-shirt, mousepad)
2. Preview your chart placed on each product
3. Adjust placement and scale
4. Add to cart and check out

## Configuration

The store connects to the chartout API. By default it points to `https://api.chartout.io`. To use a local API during development:

```python
store = chartout.Store(api_url="http://localhost:8000")
```
